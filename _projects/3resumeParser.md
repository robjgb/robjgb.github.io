description: This tool is made for resume parsing that automates the process of reading through resumes. Users upload a resume and, using a serverless GraphQL API, AWS Lambda and Amazon Textract is leveraged to query for important resume information with machine learning and stored in a DynamoDB table. A dashboard with the resumes is shown where a user can search for a specific skill or keyword.
